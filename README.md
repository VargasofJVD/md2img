md2img
======

Convert a Markdown to IMG.

## Usage

    $ go get github.com/sqeven/md2img
    $ md2img --css=main.css target.md output.png

## Requires

[wkhtmltopdf/wkhtmltopdf](https://github.com/wkhtmltopdf/wkhtmltopdf)
